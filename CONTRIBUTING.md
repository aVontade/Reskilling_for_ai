# Contributing to Reskilling for the AI Economy

We welcome contributions! This is an open project aimed at helping organizations and individuals navigate the AI transition.

## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your changes
3. **Submit a Pull Request**

## Content Contributions

- Corrections, clarifications, and additional case studies
- Translations (we'd love versions in other languages)
- Real-world examples of AI reskilling success stories
- Visual assets (charts, infographics, diagrams)

## Code Contributions

If you're improving the HTML or adding features:
- Keep it simple and accessible
- Test in multiple browsers
- Maintain the existing style where possible
